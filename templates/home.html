Weather Data Api

The format and example url to access the data of a given station at a particular date is given below.

URL format: http://127.0.0.1:5001/api/v1/station/date

URL example: http://127.0.0.1:5001/api/v1/10/1988-10-25


The format and example url to access all the data of a given station is given below.

URL format: http://127.0.0.1:5001/api/v1/station

URL example: http://127.0.0.1:5001/api/v1/10


The format and example url to access all the data of a given station in a particular year is given below.

URL format: http://127.0.0.1:5001/api/v1/yearly/station/year

URL example: http://127.0.0.1:5001/api/v1/yearly/10/1988
